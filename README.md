# Curso de Inteligencia Artificial - Asignatura de Programación

Este proyecto forma parte del **Curso de IA** dentro de la asignatura **Programación**, y tiene como objetivo mostrar un ejemplo básico de cómo crear y ejecutar un programa en **Python**.
